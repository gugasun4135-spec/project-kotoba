# Mission and Principles
## Mission
Kotoba exists to help language learners build a permanent, personal language knowledge library, reduce learning anxiety, and track real capability growth rather than temporary test scores.

## Product Principles
1. Every mistake becomes permanent structured knowledge.
2. Reduce learning anxiety, never create unnecessary pressure.
3. Core learning unit is Knowledge Node, not test questions.
4. Build a private, lifelong personal knowledge library for each user.
5. AI guides learning and analysis, it never replaces active study.
6. Mobile-first lightweight entry for instant knowledge capture anywhere.
7. Focus on long-term capability mastery instead of short-term scores.
8. All feedback delivers both data progress and gentle emotional encouragement.
9. Batch processing and inbox workflow lower the cost of recording new knowledge.
10. Design around daily 15-minute sustainable review habits.
