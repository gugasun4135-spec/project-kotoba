## User Habits
### Q1: 用户第一次什么时候打开？
After user captures their first learning material (paste OCR text / upload question image), they enter the product for first analysis.

### Q2: 用户一天会打开几次？
Minimum 2 times:
1. Daytime: Capture new knowledge / questions (Inbox input)
2. Evening: Daily 15-min review session
Potential extra opens for quick knowledge lookup.

### Q3: 我们希望培养什么长期学习习惯？
1. Instant capture of unknown knowledge at any time
2. Fixed daily 15-minute focused review cycle
3. Track personal knowledge mastery instead of test scores
4. Proactive elimination of knowledge blind spots via weakness detection engine
