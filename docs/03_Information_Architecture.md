
# Level 1 Page Navigation
Global top tab bar fixed 5 core pages:
1. Home
2. Inbox
3. Review
4. Knowledge
5. Insights
6. Profile

## Two Design Options for Inbox Placement (Undecided)
### Option A: Inbox as Independent Top Tab
Structure:
- Tab list: Home | Inbox | Review | Knowledge | Insights | Profile
- Flow: User switch tabs to enter full Inbox page, dedicated batch operation area
Advantage: Separate space for mass capture & batch analysis
Disadvantage: Extra tab switch for quick view of pending items

### Option B: Inbox as Sub Module inside Home Page
Structure:
- Single Home page contains 2 blocks: Daily Review Widget + Inbox Module
- No standalone Inbox top tab; full Inbox page accessible via "View All" button inside Home module
Advantage: One-screen overview of pending tasks without tab jump
Disadvantage: Limited space for bulk processing on home screen
